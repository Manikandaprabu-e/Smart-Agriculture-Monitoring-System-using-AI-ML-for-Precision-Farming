# AI-Based Smart Agriculture Monitoring System

## Description
This project predicts irrigation requirements using machine learning based on soil moisture and temperature data.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features
- Data-driven irrigation prediction
- Decision Tree ML model
- Real-time user input simulation
- Data visualization using scatter plot

## Outcome
The system helps in efficient water usage by predicting when irrigation is required.
